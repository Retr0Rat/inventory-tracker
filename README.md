# Inventory Tracker

A Python-based inventory management system built as part of the AIDI-2004 AI in Enterprise Systems final exam at Durham College.

## Overview

This project demonstrates core inventory management logic including stock tracking and updates, developed and version-controlled using Git and GitHub CLI.

## Features

- Track inventory items and quantities
- Update stock levels with the `updateStock` method
- Clean, single-file Python implementation

## Tech Stack

- Python
- Git & GitHub CLI
